### CINEFLEX
Aplicação front end com intuito de escolher filme, assento e sessão para assistir no cinema.

## Sobre
Requisitos implementados:
- escolha do filme
- sessão do filme
- assento de acordo com a disponibilidade
- página contendo o resumo das escolhas
